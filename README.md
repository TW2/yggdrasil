# yggdrasil
Subtitles synchronization software

Status: v1.1 alpha phase

## Abilities
- [x] Can open videos (FFMpeg)
- [x] Can open audio (FFMpeg via transcode to wave)
- [ ] Can understand VFR
- [ ] Can display full ASS
- [x] Can open/save ASS
- [x] Can synchronize an ASS event
